# Terraform AWS Infrastructure Automation

This repository uses `Project_CloudIaC.md` as the source project brief.

The website is a production-style cloud infrastructure showcase focused on Terraform, AWS, GitHub Actions, CloudWatch, Infrastructure as Code, and DevOps automation. It should use Next.js App Router, Tailwind CSS, shadcn/ui-style primitives, Framer Motion, and Magic UI-inspired animated components.
