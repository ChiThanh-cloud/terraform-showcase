# Terraform AWS Infrastructure Automation Showcase

Production-ready Next.js App Router website for a Terraform, AWS, GitHub Actions, CloudWatch, and Infrastructure as Code showcase.

## Stack

- Next.js App Router
- Tailwind CSS
- shadcn/ui-style reusable components
- Framer Motion
- Magic UI-inspired animated grid, beams, and counters
- Lucide icons

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```
